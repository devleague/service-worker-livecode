Service Worker Runtime Cache Example

Service Workers are mainly used for pre-caching or runtime caching files and/or data.
The service worker has been registered which means all the static files have been cached beforehand.
The JSON data below was fetched from a server using a normal fetch call. The data itself is now cached which means that it's still accessible with an offline connection.
